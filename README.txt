# Mini-Seite zum direkten Abspielen
Dateien:
- index.html
- vlipsy-creepy-face-jump-scare-3hEsFXt9.mp4

So nutzt du die Dateien:
1) Lade beide Dateien **gemeinsam** in denselben Ordner auf einen Hoster (z. B. Netlify Drop, GitHub Pages, Vercel oder deinen Webspace via FTP).
2) Teile die URL zur `index.html` (bei den meisten Hostern ist es einfach die Basis-URL der Seite).
